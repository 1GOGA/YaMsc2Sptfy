## Or If You Get an Error

If the remote already exists, first check:
```bash
git remote -v
```

If you see origin pointing to somewhere else, remove it:
```bash
git remote remove origin
git remote add origin https://github.com/1GOGA/YaMsc2Sptfy.git
git push -u origin master
```
